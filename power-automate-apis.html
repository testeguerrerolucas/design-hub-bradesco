APIs - Power Automate

Integre programaticamente com o Power Automate usando estas APIs REST.

Visão Geral Casos de Uso Templates Boas Práticas APIs

APIs disponíveis no Power Automate

As APIs do Power Automate permitem que você interaja programaticamente com a plataforma, automatizando a criação, gerenciamento e monitoramento de fluxos. Isso é especialmente útil para implementações em larga escala e integração com pipelines de CI/CD.

Connector API

REST

API para criar e gerenciar conectores personalizados para o Power Automate.

Endpoint: /v1/connectors
Autenticação: OAuth 2.0
Formato: JSON
Ver Documentação Ver Exemplos

Flow Management API

REST

API para criar, modificar e gerenciar fluxos programaticamente.

Endpoint: /v2/flows
Autenticação: API Key
Formato: JSON
Ver Documentação Ver Exemplos

Approvals API

REST

API para integrar com o sistema de aprovações do Power Automate.

Endpoint: /v1/approvals
Autenticação: OAuth 2.0
Formato: JSON
Ver Documentação Ver Exemplos

Business Process Flows API

REST

API para gerenciar fluxos de processo de negócio no Power Automate.

Endpoint: /v1/businessprocessflows
Autenticação: OAuth 2.0
Formato: JSON
Ver Documentação Ver Exemplos

Exemplo de código

Exemplo: Listar fluxos usando a API 

// Requisição para listar fluxos
fetch('https://api.flow.microsoft.com/v2/flows', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_ACCESS_TOKEN',
    'Content-Type': 'application/json'
  }
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Erro:', error));
Exemplo: Criar um novo fluxo 

// Requisição para criar um novo fluxo
fetch('https://api.flow.microsoft.com/v2/flows', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_ACCESS_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Meu Fluxo Automatizado',
    description: 'Fluxo criado via API',
    definition: {
      // Definição do fluxo em formato JSON
      // ...
    }
  })
})
.then(response => response.json())
.then(data => console.log('Fluxo criado:', data))
.catch(error => console.error('Erro:', error));

Autenticação

A maioria das APIs do Power Automate requer autenticação usando OAuth 2.0 ou chaves de API. Para obter um token OAuth, você precisará registrar seu aplicativo no Azure AD e solicitar as permissões adequadas.

Confira a documentação completa para detalhes sobre como obter e gerenciar tokens de autenticação.
